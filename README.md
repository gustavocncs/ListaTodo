## Lista de Tarefas
Aplicação de lista de tarefas com filtros, animações e persistência de dados.

[🔗 Acesse a demo](https://lista-todo-six.vercel.app/)

## Funcionalidades
Adicionar, concluir e excluir tarefas
Filtros por status: Todas, Pendentes e Concluídas
Contador de tarefas pendentes
Limpar todas as tarefas concluídas de uma vez
Persistência no localStorage
Animações de entrada e saída com Framer Motion

## Tecnologias
React
Vite
Framer Motion
